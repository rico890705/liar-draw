# 🎨 라이어 그림 게임

친구들과 방 코드로 접속해서 함께 즐기는 실시간 라이어 게임입니다.
한 명은 제시어를 모르는 **라이어** — 공유 그림판에 순서대로 그림을 그려가며 라이어를 찾아내세요.

Node.js + Socket.IO 로 만든 진짜 온라인 멀티플레이입니다. 각자 자기 폰/노트북으로 접속합니다.

---

## 1. 로컬에서 실행하기 (내 컴퓨터에서 테스트)

Node.js 18 이상이 필요합니다. (없으면 https://nodejs.org 에서 설치)

```bash
# 이 폴더 안에서
npm install      # 최초 1회, 필요한 라이브러리 설치
npm start        # 서버 실행
```

실행되면 터미널에 주소가 뜹니다 → 브라우저에서 `http://localhost:3000` 접속.

**같은 와이파이(집)의 친구들과 하려면**: 내 컴퓨터의 내부 IP 주소를 알아내
`http://192.168.x.x:3000` 형태로 친구들에게 알려주면 됩니다.
(맥: 시스템 설정 → 네트워크 / 윈도우: `ipconfig` 로 IPv4 주소 확인)

---

## 2. 인터넷에 배포하기 (누구나 접속 가능한 진짜 웹사이트)

집 밖의 친구들과도 하려면 무료 호스팅에 올리세요. **Render** 추천 (가장 간단).

### Render 로 배포 (무료)
1. 이 폴더를 GitHub 저장소에 올립니다.
2. https://render.com 가입 → **New → Web Service** → GitHub 저장소 연결
3. 설정값:
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - Instance Type: **Free**
4. 배포가 끝나면 `https://내앱이름.onrender.com` 주소가 나옵니다. 이 링크를 친구들에게 공유하세요!

> 포트는 코드에서 `process.env.PORT` 를 자동으로 사용하므로 따로 설정할 필요가 없습니다.
> 무료 플랜은 한동안 접속이 없으면 서버가 잠들었다가, 다음 접속 시 깨어나며 30초쯤 걸릴 수 있어요.

Railway, Fly.io, Glitch 등 다른 서비스도 같은 방식으로 동작합니다.

---

## 3. 게임 방법

1. **닉네임** 입력 후 **방 만들기** (첫 사람이 방장 👑) 또는 친구가 준 **방 코드로 참가**
2. 대기실에서 참가자는 **READY**, 방장은 **주제 선택 후 START** (3~10명)
3. 카드를 눌러 내 역할 확인 — 시민은 제시어를, 라이어는 "라이어"임을 봅니다
4. 순서대로 **공유 그림판**에 그림 추가 (총 2바퀴). 라이어는 눈치껏 그리기!
5. 토의 후 **라이어로 의심되는 사람 투표**
   - 라이어 지목 성공 → 라이어가 제시어를 맞히면 라이어 승, 틀리면 **시민 승**
   - 시민 지목 → **라이어 승**
6. 결과 확인 후 방장이 **다시하기**

├─ README.md
